# Getting Started

## Student Accounts

You will use:

- your Brandeis account for official course communication and grades,
- a GitHub account for course repositories,
- VS Code for editing Markdown and Python,
- Arduino IDE for uploading sketches,
- Python for serial communication, plotting, and modeling.


## Before the first day of class

1. Follow the instructions in [GitHub Desktop, GitHub, VS Code, and AI workflow](git-vscode-ai-workflow.md).
2. Confirm that you can sign in to GitHub.
3. Install the [Arduino IDE](https://www.arduino.cc/en/software).
4. Confirm that the installed applications open successfully.

## Reading Markdown

GitHub renders `.md` files automatically. In VS Code, open a Markdown file and
select **Open Preview to the Side**, or press **Command-K**, then **V** on a Mac.

## Course Workflow

Use the [GitHub Desktop, GitHub, VS Code, and AI workflow](git-vscode-ai-workflow.md)
page when you need to organize code, make a checkpoint, write a `README.md`, or
document how you used an AI coding assistant.
