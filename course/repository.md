# Course Repository

This website and the student course materials come from the same Git
repository. Assignments are Markdown files, so they can be read:

- as rendered pages on this website,
- directly on GitHub,
- in VS Code Markdown Preview.

Students will receive separate private repositories for their own code, data,
lab records, and submissions.

Use the [Git, GitHub, VS Code, and AI workflow](git-vscode-ai-workflow.md) page
for the minimal commands and documentation habits expected in this course.
