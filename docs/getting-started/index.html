Getting Started¶

Student Accounts¶

You will use:

  • your Brandeis account for official course communication and grades,
  • a GitHub account for course repositories,
  • VS Code for editing Markdown and Python,
  • Arduino IDE for uploading sketches,
  • Python for serial communication, plotting, and modeling.

Before Lab 1¶

  1. Install the Arduino IDE.
  2. Install VS Code.
  3. Confirm that you can sign in to GitHub.
  4. Bring a USB cable or adapter that connects your computer to an Arduino Uno.
  5. Read the GitHub Desktop, GitHub, VS Code, and AI workflow.
  6. Read the Lab 1 assignment.

Reading Markdown¶

GitHub renders .md files automatically. In VS Code, open a Markdown file and select Open Preview to the Side, or press Command-K, then V on a Mac.

Course Workflow¶

Use the GitHub Desktop, GitHub, VS Code, and AI workflow page when you need to organize code, make a checkpoint, write a README.md, or document how you used an AI coding assistant.
